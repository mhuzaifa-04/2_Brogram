# 💪 The Brogram 

**The 30 Simple Workout Program.** A minimalist, dark-themed React application designed to help you follow a structured 30-day Push/Pull/Legs training plan. Track your progress, learn new techniques, and become a lifetime gym bro.

## 🚀 Features

* **30-Day Training Plan:** A fully structured Push → Pull → Legs rotation.
* **Progress Tracking:** Unlock and track daily workouts as you progress through the program.
* **Exercise Management:** View exercise names, track weights, and log sets directly in the app.
* **Responsive UI:** A clean, dark-mode-first design styled with FantaCSS, optimized for both desktop and mobile viewing.
* **Help Modals:** Built-in tooltips and modals to guide you through exercise techniques.

## 📜 The Rules of The Brogram

To complete this program and guarantee results, you must follow these 3 simple rules:

1. **Rest:** Ensure that you are taking rest days where necessary.
2. **Reps:** Every rep is a pause rep following a **2 - 2 - 2 tempo**.
3. **Weight:** Select the maximum weight that allows you to complete the set with good form. (The first and second sets should be at 75% and 85% of your working weight used for the last two sets).
